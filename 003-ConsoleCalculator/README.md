003-ConsoleCalculator
Консольный калькулятор

Постановка задачи: 
написать программу, которая умеет выполнять следующие действия: сложение, вычитание, умножение, деление и возведение в степень. Количество чисел выбирайте произвольное (простая задача - два числа, сложная задача - количество чисел вводит пользователь).

Немного усложним задание:
1. Вначале вводится строка-выражение и выдается готовый результат, если выражение пусто, тогда запрашивается по очереди число -> действие - вывод результата

2. Числа
Целые (Int) - 1,2,3
Вещественные (Double) - 12.45, 1566666.23333333

3. Поддерживаемые операции:
+ - сложение (4)
- - вычитание (4)
* - умножение (3)
/ - деление (3)
^ - возведение в степень (2)
() - скобки, наивысший приоритет (1)

4. Строка-Выражение:
Нормальное
3+5-6/3 - операции выполняются по приоритету операций. 6/3=2 3+5=8-2=6
Сложное
3+(10-6)^2/3 - операции выполняются по приоритету и раскрытие скобок. (10-6)=4^2=16/3=5.333+3=8.333

004-OperationsOnMatrix
Консольный массив
Теория:
Матрица – это прямоугольная таблица чисел, содержащая m строк одинаковой длины n
Матрица А называется матрицей размера mxn, числа aiJ называются ее элементами,
где i показывает номер строки, а j – номер столбца. 
Числа а11, а22, а33… образуют главную диагональ
m = a.size // количество строк
n = a[0].size // количество элементов в строке

Выполнение:
1.Создадим пустую матрицу размером m x n
1.1. Запросить число m
1.2. Запросить число n
1.3. Заполнить матрицу
1.4. Вывести матрицу

2.сумму элементов главной диагонали
2.1. Через цикл for
2.2. Через цикл repeat
2.3. Выводим

3. среднее арифметическое элементов матрицы
3.1. Переводим в лист и функция .average()
3.2. Через цикл for суммируем и делим на количество элементов
3.3. Выводим

4. среднее арифметическое каждого из столбцов

5. среднее арифметическое каждого из столбцов, имеющих четные номера

6. наибольший (наименьший) элемент матрицы

7. сумму элементов 1-го столбца матрицы

8. сумму элементов строки, в которой расположен наименьший элемент матрицы (предполагается, что он единственный);

9. значение наибольшего по модулю элемента матрицы

10. сумму наибольших значений элементов ее строк

11. наибольшее из значений элементов k-ой строки

12. число отрицательных элементов в k-ой строке

13. произведение всех элементов матрицы

14. произведение квадратов тех элементов k-ой строки, которые больше 1, но меньше 3

15. произведение модулей элементов k-ой строки

16. Заменить элементы главной диагонали на 1

17. Заменить все отрицательные элементы матрицы на 0, а положительные элементы матрицы на 1

18. Заменить элементы главной диагонали на 1, а все остальные элементы на 0


--------------------------
Постановка задачи: 
Разработать программу которая умеет выводить массив M x N.

Усложнение задачи: Сделать так, чтобы можно было производить какие-либо операции над массивами.
(К примеру чисел в массиве, нахождение общего знаменателя и так далее - подобные задачи можно найти в интернете). 
Для того чтобы зачесть себе в карму усложнение задачи, решите как минимум 5 задач с матрицами.
Сделать несколькими способами.

Найти:

а) сумму элементов главной диагонали;

б) среднее арифметическое элементов матрицы;

в) среднее арифметическое каждого из столбцов;

г) среднее арифметическое каждого из столбцов, имеющих четные номера;

д) наибольший (наименьший) элемент матрицы;

е) сумму элементов 1-го столбца матрицы;

ж) сумму элементов строки, в которой расположен наименьший элемент матрицы (предполагается, что он единственный);

з) значение наибольшего по модулю элемента матрицы

и) сумму наибольших значений элементов ее строк;

к) наибольшее из значений элементов k-ой строки;

л) число отрицательных элементов в k-ой строке;

м) произведение всех элементов матрицы;

н) произведение квадратов тех элементов k-ой строки, которые больше 1, но меньше 3;

о) произведение модулей элементов k-ой строки.

Заменить:

а) элементы главной диагонали на 1;

б) все отрицательные элементы матрицы на 0, а положительные элементы матрицы на 1;

в) элементы главной диагонали на 1, а все остальные элементы на 0;
